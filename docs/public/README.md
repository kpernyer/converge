# Converge Core — Public Documentation

This directory contains public-facing documentation for the Converge Core library.

## What's Here

- **API Overview** — High-level description of the public API
- **Core Concepts** — Essential concepts without implementation details
- **Usage Guide** — How to use the library
- **Guarantees** — Behavioral guarantees and properties

## What's Not Here

Implementation details, internal architecture decisions, and design specifications are kept private. The source code for `converge-core` is not publicly available—only the compiled library is distributed.
